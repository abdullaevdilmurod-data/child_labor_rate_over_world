Введение
Этот проект посвящен исследованию проблемы детского труда в различных регионах мира. Цель работы — выявить страны с наиболее критическими показателями, проследить динамику изменений и найти корреляции с экономическими факторами.

## Инструменты
- **SQL (PostgreSQL):** Сбор и агрегация данных (использование оконных функций, подзапросов и joins).
- Выполненные работы
1. Провести расчет среднего уровня детского труда по регионам за последние 10 лет.
2. Определить топ-10 стран с наибольшим прогрессом в сокращении эксплуатации детского труда.
